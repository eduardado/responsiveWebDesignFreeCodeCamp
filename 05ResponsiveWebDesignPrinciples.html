# Responsive Web Design Certification ## Responsive web design principles Start: 10/07/2020 Pomodoros: Notas: * Better Comments extension for Visual Studio Code * Sugerencia Usar = + MAP/SINTAXIS/EJEMPLOS para saltar entre secciones * Ejemplo de colores ### Tabla de contenidos * @media (condition){css rules} conditions: * max/min-width * max/min-height * max-width * % * height * auto; --------------- Viewport units * vw * vh * vmin * vmax * viewport: el área visible. * media query: estilos css que se aplicarán sólo si se comple la condición. * max-width: 100%: la imagen nunca será más grande que el contenedor. * height: auto : se respetará el aspect ratio * pixel density * PPI: pixels per inch * DPI: dots per inch * Halth the heigth and width of a picture so it doesn't pixelate on retina's display. * Responsive typography: usar porcentajes, que se adaptan al viewport en lugar de usar em (relativo al elto padre) o px. * Viewport units * vw: viewport width * 10vw -> 10 % de la anchura del viewport * vh: viewport height * 3vh -> 3% de la altura del viewport * vmin: viewport minimun * 70vmin -> el 70 % de la dimensión más pequeña del viewport * vmax: viewport maximun(ancho o alto) * 100vmax -> el 100% de la mayor dimensión del viewport(ancho o alto)
