* float * clear * both * vertical-align #box-container { display: flex; } * display: flex * Convierte un elto enun flex container y puede usar otras propiedades flex. * Por defecto los eltos hijos se expresan en "row" * al poner display: flex; estamos el browser interpreta también flex-direction: row; * Flex-direction de css flexbox a row o column los hijos de dicho elemento se organizarán en filas o columnas respectivamente. * Flexbox ejes: https://www.w3.org/TR/css-flexbox-1/images/flex-direction-terms.svg * Justify-content * flex-wrap * wrap : los eltos extra van a otra línea/columna (dependiendo del main axis) * nowrap [default] * wrap-reverse * flex-shrink flex-grow y flex-basis se aplican a eltos hijos(flex items), el resto se aplica a elementos padre flex containers. * flex-shrink * Los eltos hijos se reducen cuando no quepan en el container. * 3 vs 1: el elto que tenga valor 3 se reducirá 3 veces más que el 1. * flex-grow : lo contrario a flex-shrink * flex-basis * Si no le damos anchura a un elto y solo altura, no aparecerá. Si usamos flex-basis: 10em si aparecerá porque hace referencia al elto padre. * Aclaración: cuántos px son 10em? * La base, si no se indica son 16px, así, 10em es diez veces la base que es 160px. * float, clear, vertical-align no funcionan con flex box. * align-self * Permite ajustar cada flex item (hijo) individualmente en lugar de en grupo (align-items) * Tiene los mismos valores que align-items * Sobreescribe elign-itms * Sirve para organizar los eltos cross axis pero uno a uno en lugar de en grupo.
